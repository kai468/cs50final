About

This web application was my final project for CS50 at Harvard University. Featuring a whole lot of different languages and ideas and really digging into the basic concepts of programming and how computers work, this class was a great learning experience, nothing like other programming courses I took before. On top of that, it's completely free so I definitely recommend checking it out!

Features

Chupochess Online features:

  • A basic chess engine implementing all the rules of the game
  • A highly responsive UI
  • An AI opponent, trained with game data from various chess grandmasters
To focus a bit more on the AI part of the project, I decided to start with a implementation allowing the user only to play with white and challenge the defending computer opponent. For further information and a live demo, feel free to have a glance at the git repository and/or project video.
